Add Survey to CRM

This module manage surveys at each stage of tasks.

User will create a survey with the questions he want to ask and will set the related_lead value of the field he want to link in question.

Under Leads and Opportunities in Sales, user can set different surveys in stages.

In the pipeline of sales,tasks are displayed in different stages.

On click of go to survey inside the form it will redirect to start survey page.

It will display the questions of survey with the answer of the linked field.

If user changes the answer,then it will be updated in the form of model also.

Under the survey page,an entry is created of survey with the state of it and answers are recorded in survey answers page.
