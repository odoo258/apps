SMS Notification

This module provides facility to send SMS notification to the mobile number.

In general settings,go to sms,install the required sms gateway.After installation configure sms gateway account and it will redirect it to the gateway configuration page.

SMS feature will be enabled or disabled under Users tab.

Under gateway configuration page you can create an account with the necessary details of the installed gateway and can test the connection afterwards.

User can send sms to group/individual member/multiple members by providing the details required and can also delete the message details by enabling auto delete.

User can also save the sms in draft state.

User can also see the list of sent sms.

User can get delivery report of all sms.

User can also create a group if he/she wants to send message to particular users.

User can also create a template to send same message to specific number of users.
