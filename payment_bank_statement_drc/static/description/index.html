Bank Statement Selection On Payments

This module added features on customer/supplier payments to allow account user to link payment with bank statement direct through payment menu or customer/supplier invoices register payment option. After selecting and validating payment, module will add bank statement line on selected bank statement.
Features are:

  • User can select bank statement through Payments menus.
  • User can select bank statement while they are on customer invoices menu and make payment through select register payment option.
  • This module will work for both customer and supplier payments.

Fields on Payment Form

Selected Fields on Payment Form

  • When User click on Add Bank statement Entry check box and make it ticked at that time Bank Statement dropdown box will be appear other wise invisible.
  • User can select bank statement form this dropdown, where user want to make entry in selected bank statement.

Register Payment Screen

  • User can register payment of those customer/supplier whose invoices are validated.
  • This registration payment transaction entry are added in selected bank statement if it has been selected.

Showing Entry On Bank Statement

  • It's show each transaction entry which is done by payment entry or register payment option from customer invoices option when user can validate customer/supplier invoices.
