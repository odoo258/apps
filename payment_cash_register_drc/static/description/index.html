Register Cash Payment

This module adds features on customer/supplier payments to allow account user to link payment with cash register direct through payment menu or customer/supplier invoices register payment option. After selecting and validating payment, module will add cash register line on selected cash register.

Payment Form And Invoice Register Payment Screen



Note:

  • When User click on Add Cash Register Entry check box,at that time Cash Register field will be visible other wise invisible.
  • User can select cash register from this dropdown, where user want to make entry in selected cash register.

Process:


  • When cacustomer/supplier whose invoices are validated,user can register payment for that invoice.
  • If user selects one cash register then this registration payment entry is added into it.

Cash Register Entry

  • When user selects cash register and validate invoice this transaction entry is shown on selected cash register.
