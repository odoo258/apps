Invoice Report Templates

Professionally Designed Odoo Invoice Reports.


Report Settings


  • With tjis app admin can edit fields like Template Base Color, Template Text Color, General Text Color, Company Name and Address Color, Customer Name and Address Color, Table Odd and Even Parity Color, Report Logo, Report Watermark Logo, Display Company Name in Bold and Display Customer Name in Bold.
  • Admin can also select company logo and water mark in the invoice.
  • Admin can choose from 6 invoice templates and can edit it.
    • Advanced
    • Creative
    • Contemprory
    • Exclusive
    • Elegant
    • Professional

  • User can choose invoie report style from invoice form and customer form also.But the first priority for invoice template set on invoice, if not then set on decided for customer and after then set on decided for company.

Note: Default invoice report template is Contemprory.


Multiple Report Printing Options

  • We can print the invoice report from three places
    • By print button in invoice form
    • Top invoices menu in invoice form
    • Top duplicate menu in invoice form



  • User can also print the duplicated invoices using this app from the top menu on invopice form.
