Account Tax Report Excel/PDF

This module allows user to print tax report in Odoo version 9 Community. We have added option to generate tax reports in excel and pdf both format.

Reports shows:

  • Taxes Paid
  • Taxes Received

Menu: Accounting/Reports/PDF Reports/Tax Report pop-up a form with filter options as follow:

  • Date Range: To get a report of particular period.
  • Target Moves: To get all entries or only posted entries.
  • Tax Reports: Option 'All taxes' will get all paid taxes and all received taxes, while 'Manual selection' option allows to select the taxes to get report of.

Reports View

Some packages is required to generate Excel Report. Please find the requirements.txt file for dependencies and run the command pip install -r path-to-file/requirements.txt
