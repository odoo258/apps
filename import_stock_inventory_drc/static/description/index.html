Inventory Create By XLS Or CSV File

This module helps to create new invenotry from Excel or csv file.

A module Describe you to create inventory from Excel or CSV file.

File supported is .xls or .xlsx or .csv

You have to install xlrd python package to use this module.

You can only import fields shown on excel sample.
