For particular user to use this module, inside (Technical Settings) access barcode label should be active

Here it shows (Dynamic Product Label) inside Sales > Product Variants > Print

After clicking on (Dynamic Product Label), a wizard will open

Inside Products(Tab), it shows selected products & Quantity which stands for howmany time it should be printed on report

Inside Fields(Tab), it asks for fields that should be printed on report with their color,font-size & sequence

Size of Barcode & Label can be set here and currency symbol position

Now, just click on print button at bottom
